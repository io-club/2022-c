# ioclub-2022-c

## 简介

- class{x} 目录可能为 ppt 源代码，没有项目源代码和 pdf，不用看

- 所有课程的代码都在 code 目录下

- 所有课程的文档，ppt 都在 docs 目录下

- 所有的视频目前都上传到了 youtube

## class 1

  - [class 1 在线 PPT](https://io-club.github.io/2022-c/class1/1)

  > - `npm install`
  > - `npm run dev`
  > - visit http://localhost:3030
  > 当然 yarn 和 pnpm 也可以，且更推荐。
